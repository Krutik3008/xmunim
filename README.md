# XMunim - Your Digital Ledger & Shop Manager

XMunim is a professional, high-performance web application designed for shop owners to manage digital ledgers, track customer balances, and maintain a transparent transaction history. It features a modern, premium "Glassmorphism" aesthetic and is fully mobile-optimized.

## ✨ Key Features

- **🚀 Performance-First**: Built with React 19 and Vite for lightning-fast load times.
- **💎 Premium UI**: Modern dark-themed design with frosted glass effects and smooth micro-animations.
- **📱 Responsive Design**: Seamless experience across mobile, tablet, and desktop devices.
- **📈 Business Insights**: Clear overview of customer ledgers, staff, and services.
- **🛡️ Secure & Reliable**: Built with modern security standards and best practices.

## 🗺️ App Roadmap (Pages & Sections)

### Main Landing Page (Home)
- **Hero**: Catchy introduction and call to action.
- **Showcase**: Visual preview of the digital ledger interface.
- **Trust Section**: Social proof and security trust badges.
- **Features Section**: Overview of the key tools provided by ShopMunim.

### Additional Pages
- **About Us**: Information about the ShopMunim mission and team.
- **Contact Us**: Get in touch for support and inquiries.
- **Legal**: Professional Privacy Policy and Terms & Conditions pages.

## 🛠️ Technology Stack

- **Frontend Core**: [React 19](https://react.dev/)
- **Build System**: [Vite 7](https://vitejs.dev/)
- **Navigation**: [React Router 7](https://reactrouter.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Styling**: Vanilla CSS (Global variables, Flexbox/Grid, Glassmorphism).
- **Quality**: ESLint for code consistency.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- [npm](https://www.npmjs.com/)

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Krutik3008/xmunim.git
   cd xmunim
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start development server:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```
